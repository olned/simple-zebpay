# simple-zebpay

## Dependencies

[node.js](https://nodejs.org/) 
[npm](https://www.npmjs.com/) 
[socket.io](https://socket.io/)

```bash
sudo apt update
sudo apt -y intstall nodejs npm
```
## Install 

```bash
git clone https://github.com/olned/simple-zebpay.git
cd simple-zebpay/
npm install
```

## Run

```bash
node zebpay_feed.js
```
